# Support

## Getting Help

For general questions, setup help, and deployment discussion, use GitHub
Discussions if they are enabled for the repository:

https://github.com/shuntps/homepage/discussions

If Discussions are not enabled, open an issue only when the topic is actionable
for the project.

## Bugs

Use the bug report issue form and include:

- Steps to reproduce.
- Expected and observed behavior.
- Environment details.
- Relevant logs or screenshots.

## Feature Requests

Use the feature request issue form and explain:

- The problem or need.
- The proposed solution.
- Alternatives considered.
- Expected benefits and potential impact.

## Security

Do not use public support channels for vulnerabilities. Follow
[SECURITY.md](SECURITY.md) instead.
